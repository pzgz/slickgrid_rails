# Slickgrid for rails

A simple wrap of [SlickGrid](https://github.com/mleibman/SlickGrid) for rails, asset pipeline enabled.

## Installation

Add this line to your application's Gemfile:

    gem 'slickgrid_rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install slickgrid_rails
